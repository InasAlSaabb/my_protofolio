# كيفية تغيير أيقونات الويب وصورة المشاركة

عند مشاركة رابط موقعك على وسائل التواصل الاجتماعي أو إرساله، تظهر أيقونة Flutter الافتراضية. لاستبدالها بصورتك الشخصية:

## الخطوات:

### 1. إعداد الصور المطلوبة

تحتاج إلى إنشاء الصور التالية من صورتك الشخصية:

#### أ) صورة Open Graph (للمشاركة على وسائل التواصل)
- **الاسم**: `og-image.png`
- **الحجم**: 1200x630 بكسل (نسبة 1.91:1)
- **المكان**: `web/icons/og-image.png`
- **الاستخدام**: تظهر عند مشاركة الرابط على Facebook, Twitter, LinkedIn, WhatsApp, إلخ

#### ب) Favicon (أيقونة المتصفح)
- **الاسم**: `favicon.png`
- **الحجم**: 32x32 أو 64x64 بكسل (مربع)
- **المكان**: `web/favicon.png`
- **الاستخدام**: تظهر في تبويب المتصفح

#### ج) أيقونات التطبيق
تحتاج إلى 4 صور بأحجام مختلفة:

1. **Icon-192.png** - 192x192 بكسل
2. **Icon-512.png** - 512x512 بكسل
3. **Icon-maskable-192.png** - 192x192 بكسل (مع خلفية شفافة)
4. **Icon-maskable-512.png** - 512x512 بكسل (مع خلفية شفافة)

**المكان**: `web/icons/`

### 2. أدوات لإنشاء الصور

يمكنك استخدام أي من الأدوات التالية:

#### أ) أدوات مجانية على الإنترنت:
- [Favicon.io](https://favicon.io/) - لإنشاء favicon
- [Canva](https://www.canva.com/) - لتصميم صورة Open Graph
- [RealFaviconGenerator](https://realfavicongenerator.net/) - لإنشاء جميع الأيقونات

#### ب) استخدام GIMP أو Photoshop:
1. افتح صورتك الشخصية
2. قصها بشكل مربع (للأيقونات) أو مستطيل 1200x630 (لـ Open Graph)
3. احفظها بالأحجام المطلوبة

#### ج) استخدام Flutter/Dart (اختياري):
يمكنك استخدام مكتبة `image` في Dart لمعالجة الصور برمجياً.

### 3. استبدال الملفات

بعد إنشاء الصور:

1. **استبدل `web/favicon.png`** بصورتك الشخصية (32x32 أو 64x64)

2. **استبدل الملفات في `web/icons/`**:
   - `Icon-192.png` → صورتك 192x192
   - `Icon-512.png` → صورتك 512x512
   - `Icon-maskable-192.png` → صورتك 192x192 (مع خلفية شفافة)
   - `Icon-maskable-512.png` → صورتك 512x512 (مع خلفية شفافة)
   - **أضف `og-image.png`** → صورتك 1200x630

### 4. بناء المشروع

بعد استبدال الصور:

```bash
flutter build web
```

### 5. النشر

إذا كنت تستخدم GitHub Pages:

```bash
flutter build web --base-href "/my_protofolio/"
```

ثم ارفع محتويات `build/web/` إلى GitHub.

### 6. اختبار النتيجة

بعد النشر، اختبر النتيجة باستخدام:

- [Facebook Sharing Debugger](https://developers.facebook.com/tools/debug/)
- [Twitter Card Validator](https://cards-dev.twitter.com/validator)
- [LinkedIn Post Inspector](https://www.linkedin.com/post-inspector/)

أدخل رابط موقعك لرؤية كيف سيظهر عند المشاركة.

## ملاحظات مهمة:

1. **صورة Open Graph**: يجب أن تكون URL كاملة (مثل: `https://inasalsaabb.github.io/my_protofolio/icons/og-image.png`)
2. **الأحجام**: تأكد من أن الصور بالحجم الصحيح لتجنب التشويه
3. **الجودة**: استخدم صور عالية الجودة (PNG أفضل من JPG للأيقونات)
4. **الخلفية**: للأيقونات maskable، استخدم خلفية شفافة أو لون موحد
5. **التخزين المؤقت**: بعد التحديث، قد تحتاج إلى مسح cache المتصفح أو استخدام أدوات التحقق المذكورة أعلاه

## مثال سريع باستخدام Canva:

1. افتح [Canva](https://www.canva.com/)
2. أنشئ تصميم جديد بحجم:
   - **1200x630** لصورة Open Graph
   - **512x512** للأيقونات
3. أضف صورتك الشخصية
4. صمم التصميم كما تريد
5. حمّل الصور وحفظها في المجلدات المناسبة

## نصائح للتصميم:

- **صورة Open Graph**: أضف اسمك وعنوانك المهني بشكل واضح
- **الأيقونات**: استخدم صورتك الشخصية بشكل دائري أو مربع مع خلفية جميلة
- **الألوان**: استخدم ألوان متناسقة مع موقعك
- **النص**: تأكد من أن النص واضح ومقروء في جميع الأحجام
